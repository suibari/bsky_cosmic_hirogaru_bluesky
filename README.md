# 超ひろがるBluesky!!

Blueskyユーザーの人間関係を「宇宙」として可視化するWebアプリ。

## スクリーンショット

![コズミックモード](docs/screenshot_cosmic.png)

## 使い方

1. トップページでBlueskyのハンドル（例: `yourname.bsky.social`）を入力する
2. 「探索」ボタンを押すと、そのユーザーのつながりが宇宙として表示される
3. 画面上部のボタンでモードを切り替えられる

## 機能

### 表示モード

2つのモードで同じデータを異なるスタイルで表示できる。

| モード | 雰囲気 | 特徴 |
|--------|--------|------|
| 🌌 コズミック | 黒背景 + 星屑 | 物理シミュレーションでノードが動きながら配置される |
| 🔵 ひろがる | 水色グラデーション | スコア順に整列した放射状配置。スライダーで表示人数を調整できる |

モード切替時はアニメーションでノードが滑らかに移動する。

### グラフの見方

- **中心ノード**: 検索したユーザー（常に中央に固定）
- **周囲のノード**: エンゲージメントスコアが高いほど大きく・近くに表示される
- **ノードにマウスを乗せる**: 名前・ハンドル・スコアがツールチップで表示される
- **ノードのサイズ**: スコアに比例（最小8px〜最大30px）

### 表示人数の調整（ひろがるモードのみ）

画面右側のスライダーで、グラフに表示するユーザー数を調整できる。

### スコアの計算方法

双方向のやり取りをもとにエンゲージメントスコアを計算する。やり取りの種類によって以下の重みがある。

| アクション | 重み |
|-----------|------|
| リプライ・引用・メンション | 高い |
| リポスト | 中程度 |
| いいね | 低い |

スコアが高いほど「深いつながり」のあるユーザーとして表示される。

## 制限・注意点

- 表示されるのはスコア上位100人まで
- 非公開アカウントはグラフに表示されない場合がある
- コズミックモードでは、グラフが数秒間動きながら最終的な配置に収束する

```bash
npm install
npm run dev        # 開発サーバー起動 (http://localhost:5173)
npm run check      # 型チェック
npm run build      # 本番ビルド
npm run preview    # 本番ビルドのプレビュー
```
